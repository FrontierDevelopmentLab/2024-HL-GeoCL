google CLI mac

1) open your terminal 

2) figure out what your system is
	uname -m

3) download the correct tar
	google-cloud-cli-darwin-arm.tar.gz

4) extract the tar.gz file
	tar -xvzf google-cloud-cli-darwin-arm.tar.gz

5) navigate to google-cloud-sdk directory
	cd google-cloud-sdk

6) show the contents, mak sure that the script install.sh is there
	ls

7) run the installation script
	./install.sh

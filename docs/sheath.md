## About
SHEAT is a ML model which predict the solar wind condition at Lagrange-1 (L1) point based the observation from SDo in Near Real Time (NRT).

## Preparing Data for SHEATH Model
SHEATH model is trained on the magnetogram, EUV and UV data from the HMI and AIA onboard Solar Dynamic Observatory.

### SOlar Dynamic Observatory Data
SHEATH uses the 26 feature vector extracted from the 12 (9 AIA & 3 HMI) different SDO observations.
